# Icon Files Required

This directory needs two PNG icon files:

1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

## Quick Options to Create Icons:

### Option 1: Online Generator
Visit https://realfavicongenerator.net/ or search "PWA icon generator"

### Option 2: Simple Colored Squares
Create two PNG files:
- 192x192px solid color (e.g., purple #667eea)
- 512x512px solid color (same color)

You can use any image editor or online tool.

### Option 3: Use icon-generator.html
Open `icon-generator.html` in your browser and download the generated icons.

## Important
The icons must be:
- PNG format
- Exact sizes: 192x192 and 512x512
- Named exactly: `icon-192.png` and `icon-512.png`
- Placed in this directory: `public/icons/`
